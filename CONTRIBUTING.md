# Contribuindo

Obrigado por contribuir com o Alfa Tech!

## Como contribuir
1. Faça um fork do repositório.
2. Crie uma branch: `feat/sua-melhoria`.
3. Faça alterações pequenas e focadas.
4. Execute `npm run lint` e `npm test`.
5. Abra um Pull Request com descrição clara.

## Padrões
- Use HTML semântico.
- Evite dependências desnecessárias.
- Mantenha o layout minimalista.

## Reporte de problemas
Use o template de issue para bugs e melhorias.
