# Código de Conduta

Este projeto adota o Contributor Covenant v2.1.

## Nosso Compromisso

Prometemos tornar a participação aberta e livre de assédio para todos.

## Padrões

Comportamentos esperados incluem respeito, empatia e colaboração.

## Aplicação

Para reportar violações, entre em contato via LinkedIn do autor listado no README.
